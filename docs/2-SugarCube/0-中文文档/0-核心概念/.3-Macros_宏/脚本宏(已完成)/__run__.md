# <<run>>

### <font style="color:#ffffff;background-color:#601BDE;">🔗</font><font style="color:#ffffff;background-color:#601BDE;"> <<run expression>></font>
功能上与<font style="color:#117cee;"><<set>></font>完全一致。设计初衷是在表达式为任意代码（而非需设置的变量）时，通过名称更好地辅助记忆——即<font style="color:#117cee;"><<run>></font>用于运行代码，<font style="color:#117cee;"><<set>></font>用于设置变量。
