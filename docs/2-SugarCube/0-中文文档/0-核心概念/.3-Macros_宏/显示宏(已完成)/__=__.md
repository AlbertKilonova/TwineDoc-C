# <<=>>

### <font style="color:#ffffff;background-color:#601BDE;">🔗</font><font style="color:#ffffff;background-color:#601BDE;"><<= expression>></font>
这个宏会将给定表达式的结果转为字符串并输出。它是<font style="color:#117cee;"><<print>></font>宏的别名（即两者功能完全相同，只是名字不同）。

:::tips
<font style="background:#DBF1B7;color:#2A4200">提示:</font>如果你只需输出 TwineScript 变量的值，只需将其直接写入普通段落文本中，它就会通过<font style="color:#117cee;">裸变量标记</font>自动显示（无需额外宏调用）。

:::

**历史：**

+ <font style="color:#df2a3f;">v2.0.0</font><font style="color:#262626;">：引入。</font>

**参数：**

+ <font style="color:#df2a3f;">expression</font><font style="color:#262626;">：一个有效表达式。有关更多信息，请参见</font><font style="color:#117cee;">表达式</font><font style="color:#262626;">。</font>

### 示例：
```plain
→ 假设 $gold 的值是 5
You found <<= $gold>> gold.             → 输出结果：你找到了5枚金币。   

→ 假设 $weight 的值是 74.6466266
You weigh <<= $weight.toFixed(2)>> kg.  → 输出结果：你的体重是74.65公斤。
```
