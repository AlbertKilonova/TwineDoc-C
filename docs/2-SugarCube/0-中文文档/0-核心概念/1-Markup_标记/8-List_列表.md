# List_列表

在行首使用星号 `*` 或井号 `#` 即可定义列表。它们是整理剧情线索或道具清单的绝佳工具（哇，那会让你的内容变得条理清晰）。

### 列表类型

* **无序列表 (Unordered List)**：使用星号 `*`。
* **有序列表 (Ordered List)**：使用井号 `#`。

### 语法示例

```SugarCube
* 这是一个无序列表项
* 它是圆点样式的

# 这是一个有序列表项
# 它会自动带上数字编号
```

> **专业小技巧**：你可以通过增加符号的数量来实现嵌套列表，比如 `**` 表示二级无序列表，这样就能做出漂亮的分级目录啦！
