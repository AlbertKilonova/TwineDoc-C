# Style_样式

通过简单的符号组合，你可以快速为文本添加加粗、斜体、下划线等视觉效果。

> **警告**：由于样式标记使用相同的符号作为开始和结束，因此**同一种样式不能嵌套在自身内部**（例如 `//强调 //内部// 强调//` 是会报错的，解析器会闹脾气哦）。

### 常用样式速查

| 类型 | 语法示例 | 渲染结果 |
| --- | --- | --- |
| **强调** | `//强调文本//` | `<em>强调文本</em>` |
| **加粗** | `''加粗文本''` | `<strong>加粗文本</strong>` |
| **下划线** | `__下划线文本__` | `<u>下划线文本</u>` |
| **删除线** | `==删除线文本==` | `<s>删除线文本</s>` |
| **上标** | `^^上标^^` | `<sup>上标</sup>` |
| **下标** | `~~下标~~` | `<sub>下标</sub>` |
