# Heading_标题

在 SugarCube 中，标题的定义非常直观。你只需要在行首使用感叹号 `!`。感叹号的数量决定了标题的级别，数量越多，级别越低（也就是重要性递减）。

### 语法与渲染对照

| 标题级别 | 语法示例 | 最终渲染为 |
| --- | --- | --- |
| 一级标题 | `!标题内容` | `<h1>标题内容</h1>` |
| 二级标题 | `!!标题内容` | `<h2>标题内容</h2>` |
| 三级标题 | `!!!标题内容` | `<h3>标题内容</h3>` |
| 四级标题 | `!!!!标题内容` | `<h4>标题内容</h4>` |
| 五级标题 | `!!!!!标题内容` | `<h5>标题内容</h5>` |
| 六级标题 | `!!!!!!标题内容` | `<h6>标题内容</h6>` |

> **提示**：虽然你可以一路写到六级标题，但通常为了排版美观，用到三级以上的情况比较少见呢（毕竟太多层级会让人眼花缭乱的）。
