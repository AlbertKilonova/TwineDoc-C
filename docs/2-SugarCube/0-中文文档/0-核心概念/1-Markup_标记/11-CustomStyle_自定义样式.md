# Custom Style_自定义样式

这是 SugarCube 最具灵活性的特性。你可以通过 `@@` 符号直接在文本中嵌入 CSS 样式、ID 或类名，实现精细的视觉控制。

> **注意**：由于标记符相同，`@@` 同样**不支持同类嵌套**（救命，别难为解析器了，它会迷路的）。

### 常用语法形式

* **内联形式**：`@@style-list;文本内容@@`
* **块级形式**：`@@style-list; \n 文本内容 \n @@`（在样式列表后紧跟换行会生成 `<div>` 容器）。

### 应用示例

| 需求 | 语法示例 | 效果说明 |
| --- | --- | --- |
| **应用类名** | `@@.warning;小心！@@` | 套用名为 `.warning` 的类 |
| **应用 ID** | `@@#header;标题@@` | 指定 ID 为 `#header` |
| **直接样式** | `@@color:red;红色文本@@` | 直接应用内联 CSS |

> **增强特性**：自 v2.31.0 起，ID 和类名可以连写（例如 `@@#hero.bold;主角@@`）。这能省下敲分号的时间，让波的码字效率更高哦！
