# Verbatim Text_纯文本

有时候你可能想展示标记代码本身，而不是让它们被解析成样式（比如在写这份教程的时候）。纯文本标记会封印 SugarCube 和 HTML 的所有魔力。

### 两种秘籍

1.  **三重双引号**：这是最常用的方式。
    `"""这里的 //斜体// 不会生效"""`
2.  **nowiki 标签**：适合包含较多特殊符号的情况。
    `<nowiki>这里的 ''加粗'' 也失效了</nowiki>`

最终它们都会被原样输出到屏幕上，就像普通文字一样乖巧。
