# Template_模板

模板是 SugarCube 里一种高级的“代词”功能。它允许你定义一个名称，然后在文本中反复引用，系统会自动帮你替换成正确的内容（简直是写多结局剧本的神器）。

### 语法规则

模板总是以问号 `?` 开头，后面跟着你定义的模板名称。

### 效果演示

假设你通过 Template API 定义了 `?他` 会根据角色性别自动返回 `他` 或 `她`：

```SugarCube
?他 总愿意倾听 ?他的 心声。
```

**渲染结果（假设当前角色为女性）：**
`她 总愿意倾听 她的 心声。`

这种动态替换能让你的文本根据变量灵活多变，再也不用手动写一大堆 `<<if>>` 宏来做判断逻辑啦（哇，那真的省力多了，感觉摸鱼时间都变长了呢）！
