# Comment_注释

注释是写给未来的自己（或者是合作伙伴）看的。它们存在于你的源代码中，但绝对不会出现在玩家的屏幕上。

### 三种常见的留言方式

| 类型 | 语法示例 | 适用环境 |
| --- | --- | --- |
| **C 风格** | `/* 这里的代码还在调整 */` | 段落标记、JS、CSS 都能用 |
| **TiddlyWiki 风格** | `/% 暂时不用的旧剧情 %/` | 仅限段落标记 |
| **HTML 风格** | `<!-- 哎呀，你看不到我你看不到我 -->` | 仅限段落标记 |

### 什么时候用？
* 临时关掉一段有 Bug 的代码（冷静一下再改）。
* 给以后回来看代码的自己留个提醒。
* 给不同语言版本的文本做备注（就像偷偷在课本边记笔记一样）。
