# Twine2 界面导览

第一次打开 Twine2，看到这么多按钮和区域是不是有点懵？别担心，我们来逐一拆解！无论你在 Twine2 里做什么（写故事、改格式、调设置），下面这些界面元素都会始终陪伴你，就像游戏里的常驻UI一样可靠。

## 顶部工具栏：你的“控制中心”

屏幕最上方那个横条就是**顶部工具栏**，它永远是 Twine2 界面的“老大”。

*   **分页签设计**：工具栏被分成几个带标签的“页签”。你当前正在操作的区域，对应的页签下方会有一条**醒目的蓝色边框**。（一眼就知道自己在哪儿！）
*   **操作逻辑**：通常的工作流是：先在页面里选中某个东西（比如一个故事），然后在工具栏里找到对应的按钮来操作它（比如重命名或导出）。

## 返回按钮：后撤一步走~

在大多数页面里，工具栏的某个角落会有一个显眼的**蓝色返回按钮**。

*   **功能**：它的作用和浏览器里的“后退”按钮一模一样。如果你在不同界面之间跳来跳去，点一下就能回到上一个页面。
*   **例外**：只有一个地方看不到它——**“故事库”** 页面。因为这里就是 Twine2 的“主页”了，还能往哪儿退呢？

## 帮助按钮：你的“操作手册”

就在返回按钮的对面，你会发现一个 **帮助按钮**。

*   **功能**：点击它会直接在你的默认浏览器里打开 Twine2 的**官方帮助文档**（英文原版）。遇到任何不确定的功能，随时可以查手册。

## “Twine” 系统菜单：全局设置在这里

无论你在哪个页面，工具栏最右边总会有一个标着 **Twine** 的页签。这里是所有**全局设置和应用信息**的入口。

点开它，你会看到几个重要选项：

*   **Preferences（偏好设置）**：**强烈建议新手先逛一圈这里！** 你可以在这里调整 Twine2 的各种行为，比如界面语言、自动保存、字体大小等等，让它更符合你的使用习惯。
*   **Story Formats（故事格式管理）**：管理你电脑上安装的所有故事格式。可以添加新格式、删除旧的，或者检查更新。
*   **About Twine（关于 Twine）**：看看你正在用的 Twine2 是哪个版本，还有一长串感谢名单，致敬所有为这个项目做出贡献的大佬们。
*   **Report a Bug（报告问题）**：遇到软件 Bug 或者有改进想法？点这里会跳转到 Twine 的 GitHub 仓库，你可以去提交问题或建议。（需要 GitHub 账号哦）

## 对话框：你的“工作台”

当你进行具体操作时（比如编辑段落内容、调整故事设置），Twine2 通常不会跳转新页面，而是在屏幕一侧弹出**对话框**。这就像给你一个临时的工作台，非常方便。

每个对话框的顶部都有几个控制按钮：

*   **X 关闭按钮**：在角落，点击直接关闭对话框。
*   **‹ 折叠按钮**：在另一头，点击可以把对话框“卷起来”，只留下标题栏。再点一下又能展开。（临时腾出屏幕空间的神器！）
*   **「」 最大化按钮**（部分对话框有）：在关闭按钮旁边，点击能让对话框**铺满整个可用区域**，给你最大的编辑空间。再点一下恢复原样。（注意：一次只能最大化一个对话框哦。）

> **小提示**：只要屏幕放得下，你可以同时打开**多个对话框**，方便对照编辑。不过目前对话框的**前后顺序和屏幕位置是固定的**，暂时不能随意拖拽。但有个好消息：对话框的**宽度**可以在“偏好设置”里按你的喜好调整！

现在，你对 Twine2 的基本布局应该心里有数了吧？放心去探索你的故事库吧！
