# Twine2 编辑器入门指南

欢迎来到 Twine2 的世界！这篇指南会带你们认识 Twine2 的操作界面和它背后的整个生态圈。如果你是第一次接触 Twine2，别担心，这里会手把手教你入门，还会给你提供更多的学习资源。

**零基础也能上手**：哪怕你完全没学过编程，也没做过游戏，Twine 对你也是友好的！这些技能点等你熟悉了之后再慢慢点亮也不迟。

## Twine 擅长做什么？

- **文字叙事**：Twine2 主打的就是文字创作，当然你也可以插入图片、音频和视频来丰富内容。

- **分支剧情**：Twine2 的界面设计得超直观，让你一眼就能看清楚故事的各种走向和分支。

- **网页发布**：Twine2 生成的是 HTML 文件，随便找个托管平台上传就能分享。别人点开链接就能玩，不用装任何软件，超级方便！

## Twine2 不太适合做什么？

这个嘛，其实很难绝对地说，因为总有创作大神能突破工具的限制。不过一般来说：

- **大量多媒体内容**：虽然可以加图片声音视频，但如果你要用的素材特别多，操作起来可能会有点别扭。这种情况可能 TweeGo 或 Ren'Py 会更顺手。

- **在线联机游戏**：想让多人同时玩你的 Twine2 故事？这实在是有点为难这个单机引擎了，不过如果你是前端技术大蛇也可以试试手搓哦（那还用啥Twine2*超小声）。

- **复杂世界建模**：如果你的作品需要构建复杂的世界模型，或者不用点击链接而是用文字输入之类的交互方式，Twine2 对你来说可能就比较费劲了。

::: tip
*创作无极限，上面说的这些限制仅仅只是Twine2不适合和不推荐的用法而已。*
:::
