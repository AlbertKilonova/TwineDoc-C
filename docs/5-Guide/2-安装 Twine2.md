# Twine2 安装指南

Twine2 提供两种使用方式，您可以根据自己的需求和设备环境进行选择：

*   **在线版**：直接在网页浏览器中使用，无需安装，开箱即用。
*   **桌面版**：下载并安装到您的计算机上，获得更稳定的本地体验。

大多数用户倾向于使用桌面版，以获得更好的性能和数据安全性。（毕竟自己的硬盘最靠谱，对吧？）在线版则适用于不便安装软件的场合，例如学校机房或临时使用的计算机。

为便于区分，本文将：
*   浏览器中直接运行的版本称为 **在线版 Twine2**。
*   安装到本地的版本称为 **桌面版 Twine2**。

## 选择适合您的版本

### 设备兼容性说明
*   **平板电脑**：目前仅支持通过浏览器使用在线版。Twine2 尚未推出官方的 Android 或 iOS 移动应用。（或许未来会有？）
*   **手机**：虽然理论上可以在手机浏览器中访问，但 Twine2 的编辑界面针对大屏幕设计，在小屏幕上操作较为困难，且大量文字输入不便，**不推荐**作为主要创作工具。（用手机写长篇故事？敬你是条汉子！）
*   **计算机**：推荐使用桌面版以获得最佳体验。在线版也可在计算机浏览器中正常使用。

### 在线版：在浏览器中使用

1.  **访问官网**：使用您的浏览器打开 [https://twinery.org](https://twinery.org)。
2.  **开始使用**：点击页面上的 **`Use in your browser`**（在浏览器中使用）按钮。

#### 重要须知与风险提示
在线版会将您的所有作品数据存储在 **当前浏览器** 的本地存储中。

::: warning 数据安全警告
请注意：如果您清除了浏览器的缓存、本地存储数据，或删除了浏览器配置文件，您在 Twine2 中创作的所有内容 **将会永久丢失**。（一键清空，悔不当初啊！）
:::

因此，如果您选择使用在线版，**强烈建议您养成定期备份的习惯**。具体备份方法请参阅[存档和导出故事](#)章节。

**进阶使用：安装为 PWA**
您可以将在线版 Twine2 “安装”为**渐进式 Web 应用程序**。这样它便可以像本地应用一样独立运行，甚至在离线时也能访问。但请注意，其数据存储机制不变，**定期备份的原则依然适用**。（PWA 也不是万能的保险箱哦！）

#### 特别注意事项：苹果设备用户

*   **iOS (iPhone/iPad)**：
    *   **必须操作**：请务必将 Twine2 网站 **添加到主屏幕**。否则，若超过7天未访问，系统可能会自动清除其存储数据，导致作品丢失。
    *   **原因**：这是 iOS 系统对 Safari 内核（包括所有第三方 iOS 浏览器）的隐私限制。（苹果的规矩，懂的都懂~）
*   **macOS Safari**：
    *   **强烈不建议**在 macOS 的 Safari 浏览器中使用 Twine2。它面临与 iOS 相同的数据清除风险，且无有效的解决办法。Twine2 会对此显示持续性警告。（珍爱作品，远离 Mac Safari！）

### 桌面版：安装到计算机

这是最推荐的使用方式。请访问 [https://twinery.org](https://twinery.org) 下载适用于您操作系统的安装包。（来，把安全感装进电脑里！）

*   **Windows**：
    1.  下载 `.exe` 安装程序。
    2.  运行安装程序，按照向导完成安装。Twine2 将被安装至“程序文件”并添加至开始菜单。

*   **macOS**：
    1.  下载 `.dmg` 或压缩包。
    2.  解压后，将 `Twine2` 应用图标拖拽至“应用程序”文件夹中。（一气呵成，优雅！）

*   **Linux**：
    1.  下载适用于您发行版的压缩包或安装包。
    2.  解压至您选择的目录即可运行。您也可以通过发行版的软件包管理器（如存在相关包）进行安装。（命令行高手请随意~）

---

> **技术细节**：在线版 Twine2 依赖于浏览器的“本地存储”技术。它类似于 Cookie，但拥有更大的存储容量（具体因浏览器而异）。更多信息可参阅[直接查看本地存储](#)章节。
