  - icon:
      src: /logos/SugarCube.svg
    title: SugarCube
    details: SugarCube（方糖）延续了 Twine 1 的传统风格，同时扩展了可用的宏指令。它比 Harlowe 功能更强大，但对于更高级的用法，有时需要更多的编程技术和开发模式知识。
  - icon:
      src: /logos/Snowman.svg
    title: Snowman
    details: Snowman（雪人）专为配合自定义 JavaScript 和 CSS 编写而设计。它没有内置的宏指令，但包含了 Underscore.js、Marked 和 jQuery 这三个 JavaScript 库。
  - icon:
      src: /logos/Chapbook.svg
    title: Chapbook
    details: Chapbook（小册子）是 Twine 2 的“第二代”故事格式，它将功能划分为“插入件”（导致文本显示）和“修饰符”（以某种方式影响文本的功能）。
